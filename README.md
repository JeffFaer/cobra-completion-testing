# A series of completion tests for the Cobra library.

This project allows to tests the different completion scripts used by the
[Cobra](https://github.com/spf13/cobra) library.


## Test fish (READY)
```
make clean
make fish
```
## Test bash (NOT READY)
```
make clean
make bash
```
## Test zsh (NOT READY)
```
make clean
make zsh
```
## Test all (NOT READY)
```
make clean
make
```
